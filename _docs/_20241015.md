- 簡単なプログラムからUSBシリアル経由でメッセージを送り、デバイスを制御する
  - Serialクラスの読み出し（ ```Serial.read()``` 系）を使う

  - plugdata (Pure Data)
    - エクスターナルオブジェクトの追加インストールが必要
    - フリー / オープンソース
  - Processing
    - ライブラリの追加インストールが必要
    - フリー / オープンソース
  - Max
    - 標準でシリアル通信用のオブジェクトがある
    - 有償、体験版あり
    - 有償版でないとパッチを保存できない
  - HTML/JavaScript
    - Chromeベースのブラウザであれば、WebSerial API が使える
